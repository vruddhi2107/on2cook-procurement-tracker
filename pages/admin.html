Pipeline Editor
View and understand each pipeline's phase sequence. Green phases are active; red-bordered phases are terminal (auto-close the request). Toggle optional phases on/off per pipeline.
ℹ How to add a new phase: Phase changes require a code + database update. Use the form below to document the new phase, then apply via a Supabase migration. The system will enforce it in the workflow track automatically once PIPELINE_WF_STEPS is updated in js/supabase-config.js.
Force Phase Override
Move any request to any phase — bypasses the normal workflow. Use only when a request is stuck or data needs correction.
⚠️ This directly modifies the database. Use sparingly. All overrides are logged as comments on the request.

A comment will be posted on the request with your name, timestamp, and reason.
